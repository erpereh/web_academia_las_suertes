# CLAUDE.md

## Proyecto
Demo web para Academia Las Suertes — academia de refuerzo escolar,
inglés e informática en el Ensanche de Vallecas, Madrid.
Objetivo: demo profesional para presentar al cliente y conseguir
el encargo de hacer la web real.

## Stack
- React 19 + Vite
- Tailwind CSS v4
- Motion (Framer Motion) para animaciones
- shadcn/ui para componentes base
- TypeScript

## Diseño — reglas obligatorias
- NUNCA usar Inter, Roboto, Arial, Space Grotesk ni fuentes genéricas
- Tipografías con personalidad y carácter (Google Fonts)
- Paleta: 1 color dominante + 1 acento fuerte + neutro. Máximo 3
- Layouts con personalidad: asimetría, overlap, generoso espacio negativo
- Animaciones con propósito: un reveal bien orquestado al cargar
  vale más que mil micro-interacciones dispersas
- Nada de gradientes púrpura, estética genérica de IA ni "AI slop"
- Mobile-first siempre
- Respetar prefers-reduced-motion en todas las animaciones
